# Notice

This project is a personalized research workflow derived from
[`juliye2025/evil-read-arxiv`](https://github.com/juliye2025/evil-read-arxiv).

The original project provided the initial idea and workflow foundation for
paper recommendation, reading, and Obsidian note organization. This repository
customizes that workflow for LLM inference systems, hierarchical state
optimization, distributed/stateful systems, runtime co-optimization,
heterogeneous hardware, KV-cache management, RAG, and continual/agent-memory
systems.

Major personalized changes include:

- renamed and reorganized the project as `llm-systems-paper-agent`;
- expanded search scope to systems/architecture/HPC/parallel/database and
  AI/ML/NLP/IR/Web/Data Mining venues;
- added `paper-ingest` for PDF/MinerU asset ingestion;
- integrated MinerU-based PDF-to-Markdown and figure extraction workflow;
- rewrote paper analysis requirements for detailed Chinese Obsidian notes;
- added Obsidian/Claudian/Codex-oriented skill installation conventions;
- adapted keywords, domains, note structure, and reading workflow to the
  author's research direction.

Please refer to the upstream repository for the original work. This repository
keeps this notice to make attribution explicit and to avoid confusion with the
upstream project.
